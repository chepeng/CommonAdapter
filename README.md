# CommonAdapter
包括：
* 通用适配器`CommonAdapter<T>`
* 多布局类型的通用适配器`MultiTypeCommonAdapter<T>`
* ExpandableListView的通用适配器`ExpandableListCommonAdapter<T>`
* 分类列表的通用适配器`SectionCommonAdapter<T>`
